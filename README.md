Fiyat Takip ve Karşılaştırma Sistemi
🧾 Proje Özeti
Bu proje, kullanıcıların farklı mağazalardaki ürün fiyatlarını tek bir platform üzerinden karşılaştırmasını ve geçmiş fiyat değişimlerini takip etmesini sağlayan bir web uygulamasıdır. Sistem, çeşitli kaynaklardan veri çekerek kullanıcıya en güncel ve en uygun fiyatları sunar.
⚙️ Kullanılan Teknolojiler
🎨 Frontend
Next.js (v16)
React (v19)
TypeScript
Tailwind CSS
Framer Motion
Recharts
Lucide React
⚙️ Backend
Node.js
Express.js
Cheerio (Web scraping)
Python (Selenium)
🔐 Güvenlik & Yardımcılar
JWT (JSON Web Token)
Axios
CORS
Helmet
🔎 Ek Özellikler
Fuse.js (akıllı arama sistemi)
Web Scraping (fiyat verisi çekme)
🚀 Kurulum ve Çalıştırma
💻 Yerel Kurulum
Projeyi indir:
Bash
git clone <repo-link>
Proje klasörüne gir
start.bat dosyasına çift tıkla
👉 Sistem otomatik olarak başlar
🌐 Sunucu Kurulumu
Backend Node.js ortamına yüklenir
Gerekli bağımlılıklar kurulur:
Bash
npm install
Veritabanı bağlantısı yapılandırılır
Python Selenium için gerekli ortam hazırlanır
Sunucu çalıştırılır:
Bash
npm start
📖 Kullanım
Kullanıcı siteye giriş yapar
Ürün araması yapar
Sistem farklı kaynaklardan fiyatları çeker
Kullanıcıya karşılaştırmalı fiyatlar sunulur
Geçmiş fiyat değişimleri görüntülenebilir
